Different Kinds of Systems

Overview:

There are many different kinds of systems. It can vary from a basic single room system to a multi room house audio system with dedicated house audio equipment.

Single Room 5.1 System

Something like this would usually be for a main living/family room. Five speakers would be hooked up/powered by a standard AV receiver. There is a left, center, and right channel along with 2 surround speakers (5 channels total). There is also a subwoofer involved hence the .1 in 5.1.

For the video, all video sources would be connected to the HDMI inputs on the receiver. To view the desired source, the correct HDMI input must be selected on the receiver, the same is true for the audio sources, except that audio sources use analog RCA connections as well as HDMI and Digital/Optical

Main Zone System + Second Audio Zone (Known as Zone 2)

In this scenario, using the same 5.1 system(listed above) as the main zone, there is another pair of speakers in a completely different area connected to the surround back/zone 2 speaker terminals on the AV receiver. This requires a receiver(Zone 2 capable) that has a secondary amp built in that is designed to power an additional pair of speakers.

Note: Most receivers require analog audio connections from the source in order to get audio to Zone2, however some new receivers allow audio from the HDMI to be used in Zone2(This eliminates the need for analog connections). For example, to get audio to Zone 2 from a cable box, there must be an analog audio cable from the audio output on the cable box to some audio input on the receiver. For the main zone, both audio and video travels through the HDMI cable.

House Audio (Whole House/Multi-Room)

House audio systems obviously vary depending on the house your working on. Lets look at a house that has a pair of speakers in three or more different rooms(Meaning, three or more audio zones). We would need a special multi-zone amplifier, these are designed to power multiple rooms(zones).

From here there would be some audio source such as a CD player or audio streaming device. There would be an analog cable connected from the audio output on the source to the audio input on the amplifier.

For more elaborate systems that include control systems, there may be multiple sources for the house. In this case a audio matrix would be used. An audio matrix is a switching device that allows for multiple sources to go out to multiple zones. For example, an 8x8 audio matrix has 8 audio inputs(Can handle up to 8 sources) and 8 audio outputs(Zones). This means that all 8 audio sources would be able to be played in any of the 8 zones on the matrix. It is important to note that from the outputs on a matrix, an analog cable is connected to the corresponding zone on the multi-zone amp.
